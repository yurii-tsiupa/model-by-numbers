# Model by Numbers

A platform for creating painting guides for physical 3D models.

## Current functionality

- Google authentication
- Protected models dashboard
- Firebase user profiles
- Firestore security rules
- Vercel deployment

## Tech stack

- Next.js
- TypeScript
- Tailwind CSS
- Firebase Authentication
- Cloud Firestore
- Firebase Storage
- Vercel

## Local development

```bash
npm install
npm run dev
